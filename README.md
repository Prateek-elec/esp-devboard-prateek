# ESP32-C3 Dev Board (KiCad)

Custom ESP32-C3 development board designed in KiCad.

## Features
- USB-C power input
- LiPo charging circuit
- 3.3V regulation
- ESP32-C3-WROOM module
- Sensors support (ex: BME280, mic amplifier, photo sensor)
- Breakout headers + boot/reset circuit

## Files
- KiCad project: `hardware/kicad/`
- Schematic PDF: `docs/schematic.pdf`
- Renders: `docs/pcb-renders/`
- Manufacturing outputs: `fabrication/`
